## current project loop

### MusicX
offline music player that just loads your files and plays them.

no extra noise, just speed + simple controls + works every time.

### ImageX
lightweight image editor for quick edits and idea testing.

not trying to be pro software, just something that gets the job done fast.

### Klipix
free video editor, kinda like capcut.

built to keep editing simple and easy without making things feel bloated.

### everything else
whatever random thing i started this week.
